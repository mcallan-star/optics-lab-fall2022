# week 1 - laser alignment

first day, mostly setup + safety (goggles, don't look into the beam, watch for reflections off rings/watches).

- HeNe, 633 nm, ~1-2 mW. logged the warmup, takes about 20 min to settle (hene_warmup.m)
- aligned the beam parallel to the table with the two-mirror "walk the beam" trick: iris near + iris far, mirror 1 fixes near, mirror 2 fixes far, repeat forever. tedious.
- beam height set to 4 inches (standard for this table)

note to self: ALWAYS get the beam level first before building anything. learned that the hard way in week 4 when nothing lined up.
